Ejercicios Intermedios

1. Cambio de Color de Texto: Selecciona un elemento en tu página y cambia su color de texto utilizando diferentes métodos: nombres de colores, códigos hexadecimales o valores RGB.

2. Fondo de Página: Cambia el color de fondo de toda la página utilizando la propiedad background-color.

3. Fondo con Imagen: Agrega una imagen como fondo a un elemento específico de tu página usando la propiedad background-image.

4. Fondo de Opacidad: Crea un bloque con un color de fondo semitransparente utilizando la propiedad rgba para el color de fondo.

5. Fondo con Patrón Repetido: Utiliza una imagen pequeña como patrón de fondo repetido en un área específica de tu página.

6. Encabezado con Fondo y Texto Blanco: Diseña un encabezado con un fondo oscuro y texto blanco para mejorar la legibilidad.

7. Fondo de Ancho Completo: Establece un fondo que se extienda por todo el ancho de la pantalla, incluso si el contenido no lo hace.

8. Texto sobre Fondo Gradiente: Aplica un fondo con gradiente y coloca texto encima de él. Experimenta con diferentes combinaciones de colores.

9. Cambiar Fondo al Pasar el Ratón: Utiliza eventos de hover para cambiar el fondo de un elemento cuando el usuario pasa el ratón sobre él.

10. Fondo Fijo al Desplazar: Crea un bloque con un fondo fijo que permanezca en su posición incluso cuando el usuario desplaza la página.
